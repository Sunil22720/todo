# TaskFlow — Premium Todo List Application

TaskFlow is a highly aesthetic, responsive, and interactive Todo List web application built with pure vanilla web technologies. It is designed to offer a premium user experience featuring modern glassmorphism panels, light/dark theme toggle, visual statistics trackers, and data persistence.

## Features

- **Rich Visual Aesthetics:** Modern glassmorphism layout, subtle animations, custom Outfit typography, and glowing gradients.
- **Dynamic Stats Dashboard:** A circular progress meter that recalculates in real-time to track completed vs. pending tasks.
- **Task Management:** Create, delete, and check off tasks.
- **Inline Renaming:** Double-click any task title or click the Edit (pencil) button to rename tasks inline instantly.
- **Metadata Tagging:** Assign categories (Work, Personal, Health, Shopping, Finance) and priorities (Low, Medium, High).
- **Sorting & Searching:** Query tasks dynamically via the search bar, filter by completion state, and sort by due dates, priorities, or creation order.
- **Persistence:** Keeps all your settings, tasks, and dark mode preference saved locally in the browser using `localStorage`.

## Getting Started

To run the application locally, you can open `index.html` directly in any web browser, or launch a simple local development server.

### Run with Python (Recommended)
```bash
python -m http.server 8000
```
Then, open [http://localhost:8000](http://localhost:8000) in your browser.
